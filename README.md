# User Satisfaction Data Analysis

This project analyzes user satisfaction data from a fictional mental wellness application.

## Objective
The goal of this analysis is to identify patterns related to user satisfaction and app usage behavior.

## Tools
- Python
- SQL

## Key Questions
- Is there a relationship between usage frequency and satisfaction?
- Does low satisfaction correlate with user dropout?
- Are there age-related patterns in satisfaction?

## Dataset
The dataset is fictional and created for learning and portfolio purposes.
